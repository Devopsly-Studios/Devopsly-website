# Elevate Academy - EdTech Platform

A premium, modern EdTech website template designed with a focus on vibrant aesthetics, glassmorphism, and smooth interactions.

## Features

- **Modern Design**: High-quality UI with modern typography (Outfit), vibrant gradients, and glassmorphism cards.
- **Responsive Navigation**: Mobile-friendly navbar with sticky scroll effect.
- **Interactive Hero**: 3D-style imagery, floating animated cards, and dynamic stat counters.
- **Course Grid**: Beautiful card layout for courses with hover lift effects.
- **Scroll Animations**: Elements reveal smoothly as you scroll down the page.
- **Stats Counter**: Animated number counters for social proof.

## Structure

- `index.html`: Semantic HTML5 structure.
- `style.css`: Vanilla CSS with custom properties (variables) for theme management.
- `script.js`: Vanilla JavaScript for interactions (Observer, Scroll, Toggles).

## Setup

1. Open `index.html` in any modern browser.
2. No build step required (Vanilla web stack).

## Customization

- **Colors**: Edit the `:root` variables in `style.css` to change the color scheme.
- **Images**: All images are sourced from Unsplash. Replace `src` attributes with your own assets.
