Estately - Premium Real Estate Platform

A modern, high-performance real estate website template designed for luxury property listings.

- Modern CSS variables & Flexbox/Grid layout
- High-quality imagery from Unsplash
- Interactive Search Bar
- Property Cards with hover effects
- Trust signals & Testimonials section

Tech Stack:
- HTML5
- CSS3 (Custom Properties)
- Vanilla JavaScript
- Font Awesome Icons
- Google Fonts (Playfair Display & Manrope)
